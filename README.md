# Pandas Employee Data Selection & Filtering 

## 📌 Overview
This project demonstrates core Pandas data selection and filtering techniques using a structured employee dataset.  
The notebook applies indexing, conditional filtering, and sorting operations to analyze employee records.

## 🎯 Objectives
The project covers:

- Creating a Pandas DataFrame from structured employee data
- Selecting specific columns
- Using `iloc` for position-based indexing
- Using `loc` for label-based indexing
- Filtering data using conditional statements
- Applying multiple filtering conditions
- Sorting data by single and multiple columns

## 🗂 Dataset Information

The dataset contains the following columns:

- EmployeeID
- Name
- Department
- Salary
- Performance Score
- Years of Service

Departments included:
- Engineering
- Sales
- Marketing
- HR

## 🛠 Technologies Used

- Python 3
- Pandas
- Google Colab / Jupyter Notebook

## 📊 Key Pandas Techniques Demonstrated

- Column selection using bracket notation
- Row selection using `iloc`
- Row selection using `loc`
- Boolean filtering
- Multiple condition filtering with `&`
- Sorting with `sort_values()`
- Multi-column sorting
